# Music.app イントロドン MVP

macOSの「ミュージック」アプリを公式プレイヤーとして使う、ローカル専用のイントロクイズです。

Apple Developer、MusicKitキー、`.env` は不要です。

## 必要なもの

- macOSのミュージックアプリ
- ミュージックアプリで再生できる曲やApple Musicライブラリ
- Node.js 18以上

## 起動

```bash
npm start
```

ブラウザで次を開きます。

```text
http://127.0.0.1:4173
```

## 初回だけ必要なこと

画面の `Music.appに接続` を押すと、macOSがAutomation許可を確認する場合があります。

許可対象は、このローカルサーバーからミュージックアプリを操作する権限です。許可するとプレイリスト取得、曲再生、停止が使えます。

## 遊び方

1. ミュージックアプリを起動して、Apple Musicにログイン済みの状態にします。
2. Web画面で `Music.appに接続` を押します。
3. 検索、プレイリスト、ライブラリのいずれかで候補曲を読み込みます。
4. `候補からデッキ作成` を押します。
5. `イントロ再生` で指定秒数だけMusic.appが再生します。
6. 曲名を入力して `判定`、または手動で `正解にする` / `不正解にする` を押します。

## メモ

- 音源の抽出や保存はしません。Music.appに再生を依頼するだけです。
- Webアプリ自体はローカルホストで動きます。
- Apple Musicカタログ全体検索ではなく、Music.appのライブラリ/プレイリスト内検索です。
